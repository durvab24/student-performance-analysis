# Student Performance Analysis and Visualization

## Overview

This project analyzes student performance data using Python and visualizes relationships between academic scores and demographic factors using Seaborn and Matplotlib.

## Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn

## Dataset

The dataset contains information about students including:

- Gender
- Math Score
- Reading Score
- Writing Score
- Parental Education

## Visualizations

### Scatter Plot
Shows the relationship between Math Score and Reading Score, colored by gender.

### Bar Plot
Displays the total Math Score grouped by Parental Education.

### Box Plot
Compares Writing Score distribution between male and female students.

### Histogram
Shows the distribution of Math Scores.

## How to Run

```bash
pip install -r requirements.txt
python student_performance.py
```

## Learning Outcomes

- Data loading using Pandas
- Data visualization using Seaborn
- Scatter plots
- Bar plots
- Box plots
- Histograms
